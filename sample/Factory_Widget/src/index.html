What is Riff??
Riff is a web based application development framework that is optimized for Samsung Mobile Devices.
Riff allows users to easily develop in Javascript, HTML, and CSS environments.
The powerful Components offered with Riff allow design elements and interactions to be used without any complicated development processes.
Methods for processing feed data such as XML are also offered, saving precious time used to develop a data processing module.
HTML Mark-up
Riff marks up the index.html file to create a Web Application structure. The difference between previous Web Development is that with Riff, you don’t have to mark up everything shown on screen, you just have to mark up the structure of the Components being used.
The following are automatically marked up by Riff.
UI Component
Provides modularized Components that have all the functionality you need for you Web Application.
You can simply develop a Web Application by combining Components.
Making full use of Components is the way to use Riff.
Touch Event
Provides Event that makes Buttons, Lists and certain Objects to respond to Touch.
Although similar to the Mouse Event, API for Touch exclusive events such as Flick and Swipe are offered.
